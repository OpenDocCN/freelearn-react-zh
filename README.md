## 下载

### Docker

```
docker pull apachecn0/freelearn-react-zh
docker run -tid -p <port>:80 apachecn0/freelearn-react-zh
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g freelearn-react-zh
freelearn-react-zh <port>
# 访问 http://localhost:{port} 查看文档
```
